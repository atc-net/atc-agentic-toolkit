# Playwright Plugin

Browser automation and testing skills using Playwright for website exploration and test generation.

## Overview

This plugin provides skills and an MCP server for browser automation with Playwright.

## MCP Servers

This plugin bundles the Playwright MCP server that starts automatically when the plugin is enabled.

### Playwright

Browser automation server for web testing, scraping, and interaction.

- Navigate, click, fill forms, take screenshots, and execute JavaScript in a browser context

Requires: Node.js

Source: [@playwright/mcp](https://www.npmjs.com/package/@playwright/mcp)

## Skills

### playwright-cli

Full browser automation via the `playwright-cli` command line. Covers all CLI commands (navigate, click, fill, screenshot, etc.), plus 13 reference guides for accessibility testing, responsive testing, authentication patterns, CI/CD integration, request mocking, session management, storage state, tracing, video recording, test generation, and performance measurement.

### playwright-explore-website

Explore a website interactively using Playwright MCP for testing purposes.

### playwright-generate-test

Generate Playwright tests based on a scenario using Playwright MCP.

## Requirements

- Node.js
