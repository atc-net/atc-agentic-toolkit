# 🎉 ATC Plugin

Skills for the [atc-net](https://github.com/atc-net) ecosystem — helping developers work with atc-net libraries for REST API generation, WPF development, and more.

## Skills

### rest-api-source-generator

Guides developers through OpenAPI → C# code generation using `Atc.Rest.Api.SourceGenerator`. Covers CLI tool usage, marker file configuration, security patterns, rate limiting, caching, resilience, versioning, TypeScript client generation, analyzer rules, and migration from the old generator.

> ⚠️ **Important:** The old `atc-rest-api-generator` CLI is **OBSOLETE**. Only `atc-rest-api-source-generator` should be used.

**Usage:**

```
/atc:rest-api-source-generator
```

### wpf

Guides developers through building WPF applications with `Atc.Wpf` (160+ controls, theming, 4-tier architecture) and `Atc.XamlToolkit` (MVVM source generators for WPF/WinUI/Avalonia).

**Usage:**

```
/atc:wpf
```

## Requirements

- .NET 10 SDK
- C# 14
